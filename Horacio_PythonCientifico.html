Librería Core (5/5): Pandas¶

  • RESUMIENDO: es un excel, pero mejor, mucho mejor.
  • Implementa tablas numricas: columnas, filas y celdas
  • Muy eficiente y muy completa
  • Herramientas: Filtrado, Join, GroupBy, manejo de datos faltantes, Serie de tiempo, etc.
  • La recomendada si solo vas a analizar datos

¿Qué tengo que instalar?¶

  1. Prueba con otros sistemas operativos (GNU Linux por ej, Debian, Ubuntu, ArcLinux)
    • Puedes tener dos sistemas operativos, perdida mnima de compatibilidad, etc
  2. Cmo instalo Python?
    • Camino 1: bajar Python de www.python.org Recomiendo python 3.x. Libreras (Numpy, pandas, etc.) pueden instalarse va Pip.
    • Camino 2: Recomiendo Anaconda (Distribucin de Python) Libreras se instalan conda o pip
